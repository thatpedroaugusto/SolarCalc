# Especificações do Projeto

A Calculadora Solar é uma aplicação web que mostra a quantidade de dinheiro você conseguiria economizar caso fizesse a instalação de um número x de painéis solares na sua residência. Para fazer o cálculo, basta informar seu Estado, Cidade, e valor atual da sua conta de energia.

# Resultados

Estas são as informações apresentadas no resultado:

- [ ] Quantidade de Painéis
- [ ] Potência do painel
    - [ ] Permitir usuário alterar a potência padrão.
- [ ] Área necessária para instalação.
- [ ] Economia média anual.
- [ ] Geração de energia por mês.
